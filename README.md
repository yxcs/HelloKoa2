# HelloKoa2
Koa2 程序构建，包含错误处理、测试、中间件开发、日志书写的简单的项目
1. **cross-env**  设置运行环境  
2. **log4js**  打印日志，使用的是1.*版本
3. **mocha** **chai**  **supertest**  测试

#### [详情解析](http://www.jianshu.com/p/6b816c609669)


#### 运行  
``` bash
git clone https://github.com/yxcs/HelloKoa2  
cd HelloKoa2  
npm install  
npm start
```
